| 项目 | 要求 |

| 操作系统 | macOS / Windows |
| 浏览器 | Chrome 90+ / Edge 90+ |
| Node.js | v18+ |

### 5.3 测试通过标准

1. **P0 级测试用例**: 100% 通过
2. **P1 级测试用例**: 90% 以上通过
3. **严重缺陷**: 0 个
4. **一般缺陷**: 不超过 5 个

### 5.4 缺陷严重等级

| 等级 | 定义 | 处理要求 |
|------|------|----------|
| 严重 | 功能无法使用，程序崩溃 | 必须修复 |
| 高 | 主要功能受影响，有变通方案 | 必须修复 |
| 中 | 次要功能异常，不影响主流程 | 建议修复 |
| 低 | 界面细节问题 | 可选修复 |
