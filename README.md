# Excel 离线浏览器工具

一个纯前端实现的 Excel 处理工具，无需安装软件、无需上传服务器，直接在浏览器中完成 Excel 文件的拆分与合并操作。

## ✨ 功能特性

- 📊 **按工作表拆分** - 将一个 Excel 文件中的多个工作表拆分为多个独立文件
- 📑 **按列拆分（横向）** - 按指定列的值分组，将数据拆分成多个文件
- 📋 **按列拆分（竖向）** - 将多个数据列拆成多个文件，每个文件保留固定列和一个数据列
- 🔗 **多文件合并** - 将多个 Excel 文件中的工作表合并到一个工作簿
- 📦 **工作表数据合并** - 将多个工作表的数据合并为一张总表

## 🚀 快速开始

无需安装，直接在浏览器中打开 `excel.html` 即可使用。

```bash
# 克隆仓库
git clone https://github.com/yourusername/excel-offline-tool.git

# 进入目录
cd excel-offline-tool

# 用浏览器打开
open excel.html
```

或者直接将 `excel.html` 文件拖到浏览器窗口中。

## 📖 使用说明

1. **选择处理模式** - 点击顶部按钮切换拆分或合并模式
2. **上传文件** - 支持点击选择或拖拽上传 Excel 文件
3. **配置选项** - 根据模式选择需要处理的列或工作表
4. **生成结果** - 点击处理按钮，等待完成
5. **下载文件** - 单文件直接下载，多文件打包为 ZIP 下载

## 🔒 隐私说明

- ✅ 所有文件仅在本地浏览器中处理
- ✅ 不会上传任何数据到服务器
- ✅ 无需登录，无需注册

## 🛠️ 技术栈

- 原生 HTML5 + CSS3 + JavaScript
- [SheetJS](https://github.com/SheetJS/sheetjs) - Excel 文件读写
- [JSZip](https://github.com/Stuk/jszip) - ZIP 文件生成

## 📋 支持的格式

| 格式 | 读取 | 写入 |
|------|------|------|
| .xlsx | ✅ | ✅ |
| .xls | ✅ | ❌ |
| .zip | - | ✅ |

## ⚠️ 使用限制

- 最大单文件大小：20MB
- 最大总文件大小：100MB
- 最大文件数量：50
- 最大工作表数量：200
- 浏览器：Chrome、Edge（支持 Windows 7+）
