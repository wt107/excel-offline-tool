# Excel 离线浏览器工具

一个纯前端实现的 Excel 处理工具，无需安装软件、无需上传服务器，直接在浏览器中完成 Excel 文件的拆分与合并操作。

---

## ✨ 功能特性

| 功能 | 说明 |
|-----|------|
| 📑 按工作表拆分 | 将多工作表 Excel 拆分为多个独立文件 |
| 📊 按列拆分（横向） | 按指定列的值分组，将数据拆分成多个文件 |
| 📋 按列拆分（竖向） | 将多个数据列拆成多个文件 |
| 🔗 文件合并 | 将多个 Excel 文件合并到一个工作簿 |
| 📦 工作表数据合并 | 将多个工作表的数据合并为一张总表 |

---

## 🚀 快速开始

**无需安装，双击即用**

### 方式一：直接下载使用
1. 下载 `excel.html` 文件
2. 双击用浏览器打开即可使用

### 方式二：克隆仓库
```bash
git clone https://github.com/wt107/excel-offline-tool.git
cd excel-offline-tool
open excel.html
```

---

## 📖 使用说明

1. **选择处理模式** - 点击顶部按钮切换拆分或合并模式
2. **上传文件** - 支持点击选择或拖拽上传 Excel 文件
3. **配置选项** - 根据模式选择需要处理的列或工作表
4. **生成结果** - 点击处理按钮，等待完成
5. **下载文件** - 单文件直接下载，多文件打包为 ZIP 下载

## ⚠️ 使用限制

- 最大单文件大小：**20MB**
- 最大总文件大小：**100MB**
- 最大文件数量：**50**
- 最大工作表数量：**200**
- 推荐浏览器：Chrome、Edge

---

## 🛠️ 技术栈
